# Deploy

## Command

```shell
wget https://raw.githubusercontent.com/pengqi-bc/barkisnet-deploy/master/testnet/deploy.sh -O deploy.sh && sh deploy.sh [moniker] [miniGasPrice]
```

Example command:

```shell
wget https://raw.githubusercontent.com/pengqi-bc/barkisnet-deploy/master/testnet/deploy.sh -O deploy.sh && sh deploy.sh myBarkisNode 0.01ubarkis
```
